# Student Grade Prediction MLOps Project

## Project Overview

This project predicts student performance/grade based on study-related features.  
It contains a FastAPI backend for ML model inference and a Streamlit frontend for user interaction.

## Tech Stack

- Python
- FastAPI
- Streamlit
- scikit-learn
- Docker
- Docker Compose
